# 抖音博主采集器

双击 `抖音博主采集器.cmd`，或桌面快捷方式 `抖音博主采集器`。

## 使用流程

1. 输入抖音分享短链、博主主页链接，或 `sec_uid`。
2. 选择输出目录。
3. 首次使用先点 `打开/登录浏览器`，在打开的 Chrome 里登录抖音并完成验证码。
4. 回到工具，点 `开始抓取`。
5. 如果勾选 `抓取后转写口播`，会继续把视频口播转成文字。

## 输出文件

采集 CSV 会包含：

- 标题/描述
- 作者
- 点赞、收藏、评论、分享
- 封面 URL
- 视频 URL
- 视频宽高、大小、格式
- 发布时间

转写会输出到 `transcripts` 子目录：

- `*_transcripts.csv`
- `*_segments.json`

## 注意

- 不会长期保存视频文件；转写时只临时读取视频并抽音频，结束后删除临时文件。
- 转写默认使用本地 `faster-whisper tiny`，速度较快但可能有少量错字。重要爆款可改用 `base` 或 `small`。
- 抖音页面显示的作品数和接口实际返回数可能不同，取决于登录态、可见性、审核状态和平台接口限制。
